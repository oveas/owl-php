static OWLinstaller::addTablePrefix ( $  q,
$  prefix 
) [static, private]

Check an SQL statement and lok for table names. When a tablename has been found, the table prefix will be added.

The following SQL queries are supported by the regular expressions (with and without backticks):

  • CREATE [TEMPORARY] TABLE [IF NOT EXISTS] `tblname`
  • ALTER [IGNORE] TABLE `tblname`
  • DROP [TEMPORARY] TABLE [IF EXISTS] `tblname`
  • INSERT [insert type] [IGNORE] [INTO] `tblname` (...)
  • CREATE [type] INDEX [name] [using type] ON `tblname` (...)
    Parameters:
    [in]$qSQL query
    [in]$prefixPrefix, or null to use the default
    Returns:
    SQL statement with the prefixed tablename
    Todo:

    Add support for UPDATE, DELETE, RENAME and other statements that *might* occur in SQL install files

    Handle constraints ([...] REFERENCES `tblname` (...))
