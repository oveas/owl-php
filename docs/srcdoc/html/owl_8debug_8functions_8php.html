OWLdbg_add ( $  level,
&$  var,
$  name = 'Unknown variable',
$  shiftUp = 0 
)

Add information to the debug array. OWLdbg_traceCall() is called to find out from where this function was called.

Parameters:
[in]$levelDebug level. This is a single bit, the information will only be added if this bit is true in the debug config setting.
[in]$varVariable that will be dumped
[in]$nameInformation about the variable, e.g. the name.
[in]$shiftUpNumber of levels to shift up (e.g. 1 for the caller's caller). Default 0
