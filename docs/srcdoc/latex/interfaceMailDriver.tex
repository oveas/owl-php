Send the email as it has been composed 
\begin{DoxyParams}[1]{Parameters}
\mbox{\tt in}  & {\em \$mail} & Indexed array defining the mail, with the following keys:
\begin{DoxyItemize}
\item from: Email address from which the mail is send
\item recipients: Array with all recipients; will not appear in the header
\item reply\_\-to: Displayable reply-\/to address (optional)
\item sender: Displayable email address of the sender
\item to: Array with displayable addresses as they appear in the To header (optional)
\item cc: Array with displayable addresses as they appear in the CC header (optional)
\item date: Date of the email (optional)
\item subject: Subject of the email (optional)
\item headers: Additional headers (optional)
\item body: Body of the email message 
\end{DoxyItemize}\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
: I don't know if there's a common name for mail addresses in the format \char`\"{}Personal name $<$email@example.org$>$\char`\"{} (and older vaiants). Since the RFC's (mainly 5322) mention this syntax for display purposes, I call this a \char`\"{}displayable email address\char`\"{} which differs from an \char`\"{}email adress\char`\"{} (of which the syntax is just \char`\"{}email@example.org\char`\"{}). 
\end{DoxyNote}
