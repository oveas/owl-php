\subsection{Detailed Description}
Timer handler. 

This abstract class sets up timers that can be used to check the runtime of both the total and parts.

To enable timers, the constant OWL\_\-TIMERS\_\-ENABLES must be defined by the application {\itshape before\/} \doxyref{OWLloader.php}{p.}{OWLloader_8php} is included, e.g.: 
\begin{DoxyCode}
 define ('OWL_TIMERS_ENABLED', true);
 require (OWL_ROOT . '/OWLloader.php');
\end{DoxyCode}


Timers can be started anywhere in the code. When timers are not enabled, it won't have any effect. Starting timers can be done with \doxyref{OWLTimers::startTimer}{p.}{classOWLTimers_a3e9df1345159263a223da62b775140a8}('name'). Later in the code they can be stopped with \doxyref{OWLTimers::stopTimer}{p.}{classOWLTimers_a1ba41542f07dff9ae48f15fb60d028c7}('name'), where 'name' must be unique.

At the end of the run, the total running time of the application will be displayed, end the running time of the individual timers (the part between \doxyref{startTimer()}{p.}{classOWLTimers_a3e9df1345159263a223da62b775140a8} and \doxyref{stopTimer()}{p.}{classOWLTimers_a1ba41542f07dff9ae48f15fb60d028c7}) with their percentage of the total time.

Timers that are not stopped, will be timed until the application rundown. \begin{DoxyAuthor}{Author}
