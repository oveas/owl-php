FormHandler::cleanString ( $  input,
$  allow = null,
$  content = null,
$  remove_comment = true 
)

Remove all tags from an input string

Parameters:
[in]$inputInput string to clean
[in]$allowarray with tags that should not be removed. These can be in a regular expression format, e.g. array('b', 'h[1-6]') to allow the <b> tag and <h1> to <h6>.
[in]$contentarray with tags that should completely be removed, so including all contents between the opening and closing tag, e.g. '<script>some code</script>'. Default is array('script','style')
[in]$remove_commentWhen true (default), content blocks will completely be removed. Set to false to allow the content of a comment block. The tags themselves ('<!--' and '-->') will always be removed.
Returns:
The string with all tags removed
