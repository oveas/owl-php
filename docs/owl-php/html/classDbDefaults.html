DbDefaults::isRetryable ( $  _errorCode)

Check is the given error code is a retryable error (e.g. table locked, server starting etc) and advice how long to wait before retry. Retries can be enabled at driver level, by default retries are never possible.

Parameters:
[in]$_errorCodeThe errorcode
Returns:
Adviceable time to wait for a retry in milliseconds, or 0 if no retry is possible
