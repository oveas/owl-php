Detailed Description

These bitmaps define the layout of status codes. They are used to extract information from the code

Enumeration Type Documentation

enum OWL_APPLICATION_PATTERN

Bits 1-8 define the application. Application identifiers with the first bit set (0x40 - 0xff) are reserved for Oveas. 0xff is the OWL Identifier.
